## T113_IO_SPI



### SPI配置

数据宽度：16bit

先行位：MSB

SPI模式：CPOL：Low，CPHA：0



### 初始化

主机发送：0xA501开始初始化

从机回复：0xA502确认

主机发送初始配置

从机回复：0xA50F初始化完成

| 设备 | 开始指令 | 确认回复 | 模拟量采集配置 | 开关量输出配置 | 配置包包尾 | 成功回复 |
| ---- | -------- | -------- | -------------- | -------------- | ---------- | -------- |
| 主机 | 0xA501   | ----     | 0xXXXX         | 0xXXXX         | 0xA503     | ----     |
| 从机 | ----     | 0xA502   | ----           | ----           | ----       | 0xA50F   |

----：为任意数据（默认0xFFFF）

0xXXXX：为配置数据



### 开关量输入

主机发送：0xA511

从机回复：开关量输入数据



### 开关量输出

#### 开关量输出查询

主机发送：0xA521

从机回复：开关量输出配置

#### 开关量输出配置

主机发送：0xA523

主机发送配置数据，从机返回0xA524

主机发送0xA525，从机回传配置数据正确表示成功。

| 设备 | 配置指令 | 配置数据 | 数据包尾 |
| ---- | -------- | -------- | -------- |
| 主机 | 0xA523   | 0xXXXX   | 0xA525   |
| 从机 | ----     | 0xA524   | 0xXXXX   |



### 模拟量输入

#### 模拟量数据包格式

| bit15          | bit14                          | bit13-bit0          |
| -------------- | ------------------------------ | ------------------- |
| 1bit（错误位） | 1bit（0：电压模式1：电流模式） | 14bit（模拟量数据） |

| 从机 | ----     | 0xA534   | 0xXXXX   |
